# 👋 Hi, I'm Harun

I'm a software engineer based in Copenhagen (I can relocate), building AI-powered products and tools at [`Omika`](https://omika.dk). I've been shipping software for a long time, and these days I split my time between my own products and select consulting work.

I'm the maker of [`askpdfs.io`](https://askpdfs.io), a RAG-based PDF chat SaaS I've been iterating on for ~3 years. It's where I do most of my hands-on work with retrieval pipelines, model routing, and the messy reality of running an AI product at scale - balancing latency, cost, and quality across providers like Claude, GPT, and Gemini.

I'm also building [`Veya`](https://veya.app), my own personal iOS fitness coaching app written in Swift/SwiftUI. It pulls signal from HealthKit, Strava, and Hevy and uses Gemma E2B as the AI layer to turn that into actually useful coaching - built for me who wants structured feedback, not generic advice.

On the side I run [`BilligBid`](https://billigbid.dk), a Danish food waste app on iOS and Android that helps people find discounted groceries before they're thrown out tailored towards people who have a hard time making ends meet, and [`dagensland.dk`](https://dagensland.dk), which is a Wordle/Wørdle inspired geography game which has grown to ~3,000 unique monthly visitors.

I also publish smaller open-source utilities like [`exifm`](https://www.npmjs.com/package/exifm) - a tiny Node.js utility for safely setting JPEG EXIF orientation and optionally stripping GPS metadata.
